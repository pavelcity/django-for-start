# Создаем django проект 


## #Подготовка

### Узнать версию python
=== "Mac/Linux"
    ```
    python3 -V
    ```
=== "Windows"
    ```
    python -V
    ```


### Список установленных пакетов pip - `в глобальном окружении`
```
pip list
```


## #Создание проекта

### Создать  виртуальное окружение
=== "Mac/Linux"
    ```
    python3 -m venv venv 
    ```
=== "Windows"
    ```
    python -m venv venv 
    ```

### Активировать  виртуальное окружение
```
source venv/bin/activate
```
